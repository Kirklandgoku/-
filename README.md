# 派車小工具

## 線上使用
點 `index.html` 或部署後的網址即可使用。

## 部署到 Netlify（免費、不用註冊）

1. 到 https://app.netlify.com/drop
2. 把整個 `deploy` 資料夾拖進去
3. 5 秒拿到網址，給朋友即可

## 部署到 GitHub Pages

1. 到 https://github.com/new 建倉庫
2. 上傳 `deploy/index.html`
3. Settings → Pages → Branch: main → Save
4. 拿到 `https://你的帳號.github.io/倉庫名/`
